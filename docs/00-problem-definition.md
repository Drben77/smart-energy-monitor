# Problem Definition — v1.0

## 1. Problem statement
Homeowners and renters have no way to see how much electricity an individual
appliance consumes or costs to run. They only see a total bill each month,
which makes it impossible to identify high-consumption devices.

## 2. Proposed solution
A plug-in smart energy monitor that sits between a wall outlet and an appliance,
measuring real-time energy data and displaying it locally and via a web dashboard.

## 3. Target user
Homeowners and renters who want to understand the energy use of a single appliance.

## 4. What it measures and displays
- Voltage (V, RMS)
- Current (A, RMS)
- Power (W, real power)
- Energy (kWh, accumulated)
- Estimated cost ($, based on tariff rate)

## 5. How the user interacts
1. Plug device into wall outlet.
2. Plug appliance into the device.
3. Monitoring starts automatically — no configuration required for basic use.
4. View live data on the OLED screen or the web dashboard (live values, graphs,
   daily/weekly history).
